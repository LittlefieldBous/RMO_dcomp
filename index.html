Design Comp ~ By Amy Littlefield-Bousamra

This is a fictional movie website. The main purpose is to promote the upcoming movie, Queen Elizabeth Returns. In order to help market the movie this official movie website will present the video gallery/trailer, teaser, additional video clips, a short synopsis and brief bios of the cast. For this project I wanted to create a one page site using a parallax scroll effect. The navigation bar is fixed and does not scroll so one can quickly navigate back home or elsewhere.
